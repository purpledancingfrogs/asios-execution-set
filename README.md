# ASIOS Execution Set

Canonical public execution set for ASIOS / Aureon.

## Requirements
- Python 3.11+
- Git

## Usage
Run an executor:
\\\
python asiosctl.py run <executor>
\\\

Available executors:
- decision-decomposition
- qmck
- recursive-paradox-governance

## Validation
Executors are validated automatically via GitHub Actions.
